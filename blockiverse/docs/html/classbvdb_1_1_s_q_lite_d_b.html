Detailed Description

Manager class for SQLite3 database and connections.

Manages specification of database file, creating connections and shutting down.

Cacheing system utilizes prepared statements to remember SQL statements on first use to prevent the overhead of subsequent compilations of SQL statements so long as class user utilizes binding semantics versus inline parameters.
